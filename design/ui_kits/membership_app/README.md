# UI Kit — אפליקציית מועדון כיוונים

אפליקציה אינטראקטיבית במובייל, חמישה מסכים, RTL בעברית, סגנון נאו-ברוטליסטי.

## להפעיל

פתחו את `index.html` בכל דפדפן מודרני. הכל מקומי חוץ מ-React, Babel וGoogle Fonts.

## קבצים

| קובץ | מה הוא עושה |
|---|---|
| `index.html` | טוען React + Babel + iOS frame + כל קבצי ה-JSX, ועוטף את `<App />` במסגרת אייפון. |
| `ios-frame.jsx` | מסגרת iOS 26 (בזל + dynamic island + home indicator). |
| `components.jsx` | אטומים — `Icon`, `Avatar`, `AvatarStack`, `Button`, `Badge`, `TierChip`, `Sticker`. |
| `data.jsx` | נתוני דמה — `MEMBER`, `EVENTS`, `PERKS`. |
| `screens-home.jsx` | מסך הבית + רכיבי `EventRow` + `MiniPerkTile` + `SectionHead`. |
| `screens-card.jsx` | מסך כרטיס החבר — הקליקו על הכרטיס כדי לסובב ל-QR. |
| `screens-other.jsx` | אירועים, הטבות, פרופיל, ו-EventSheet (גיליון תחתון). |
| `app.jsx` | מצב גלובלי, סרגל ניווט תחתון, ניתוב גיליונות, טוסטים. |

## מסכים

1. **בית** — ברכה, אירוע "הערב", התקדמות שלב, השבוע, גלילה אופקית של הטבות, CTA לפתיחת כרטיס.
2. **אירועים** — רשימה מסוננת (שבוע / חודש / הכל), CTA "תארחו משהו".
3. **כרטיס** — כרטיס מתהפך. הקליקו כדי לראות את ה-QR. שלוש פלקות סטטיסטיקה.
4. **הטבות** — חיפוש + רשת 2 עמודות. אריח מומלץ במגנטה. אריחים נעולים עם תווית "מייסדים".
5. **אני** — כותרת פרופיל + שורות הגדרות עם toggle עובד להתראות.

הקליקו על אירוע ← נפתח גיליון תחתון עם פרטים מלאים. אישור הגעה מציג טוסט ונשמר במצב.

## הערות

- נתוני דמה בלבד; אין רשת.
- ה-QR סינתטי — חזותי בלבד, לא ניתן לסריקה.
- מצב הטאב נשמר ב-`localStorage` תחת `kvn-tab`.
